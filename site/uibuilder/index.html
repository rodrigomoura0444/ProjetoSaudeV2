    UI Builder — ProjetoSaudeV2

    1. Overview

    The uibuilder component provides the visual layer of the ProjetoSaudeV2 system.
    It enables users to monitor patient data in real time through a clean, responsive, and interactive web interface.

    2. Purpose

    The user interface is designed to display real-time vital sign information for multiple patients, including:
    - Heart Rate (BPM)
    - Temperature (°C)
    - Blood Pressure (mmHg)
    - Oxygen Saturation (SpO₂)

    Each patient is represented as an individual card, which updates automatically whenever new data is received from Node-RED.

    3. Files Used

    File Description
    index.html Defines the overall structure and layout of the dashboard (HTML).
    index.css Controls design, colors, spacing, and visual presentation (CSS).
    index.js Manages real-time data communication and updates (JavaScript).

    These files are located in the following directory:
    ProjetoSaudeV2/ui/

    4. Example Layout

    +----------------------------------------------------+
| ProjetoSaudeV2 — Real-Time Patient Monitoring      |
|----------------------------------------------------|
| [ Patient A ] [ Patient B ] [ Patient C ] [ ... ]  |
| Heart Rate: 78 bpm  | Temp: 36.6°C | SpO₂: 97%     |
| Blood Pressure: 120 / 80 mmHg                      |
+----------------------------------------------------+

    The interface design uses soft colors, rounded components, and a modern layout suitable for medical and educational applications.

    5. Real-Time Connection

    • The UI connects directly to Node-RED through the uibuilder socket.
    • When new data is received by the backend, the interface updates automatically.
    • No manual refresh is required — all data changes are reflected instantaneously in the browser.

    6. Customization

    The frontend can be modified and extended by editing the following files:

    • index.css — Customize colors, alignment, and overall design.
    • index.js — Adjust update frequency or implement custom alerts (e.g., SpO₂ < 90%).
    • index.html — Add new patient sections, widgets, or graphical elements.

    7. Summary

    The uibuilder dashboard transforms live data from the backend into a dynamic and user-friendly web interface.
    It serves as the visual bridge between automation and human interaction, offering simplicity, responsiveness, and real-time performance within the ProjetoSaudeV2 ecosystem.
